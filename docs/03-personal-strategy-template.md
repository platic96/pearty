## 3. 투자 전략 선택

### 주요 전략 선호도 체크

아래 전략 중 관심 있는 것에 체크하세요 (복수 선택 가능):

- [ ] **가치투자** — 저평가된 우량주를 장기 보유
- [ ] **성장투자** — 높은 성장 잠재력을 가진 기업에 투자
- [ ] **인덱스투자** — 시장 전체에 분산 투자 (ETF)
- [ ] **모멘텀투자** — 상승 추세의 종목에 편승
- [ ] **배당투자** — 안정적인 배당 수입 추구
- [ ] **퀀트투자** — 데이터 기반 체계적 투자
- [ ] **스윙 트레이딩** — 단기 가격 변동 활용
- [ ] **자산배분** — 여러 자산군에 분산하여 리스크 관리
